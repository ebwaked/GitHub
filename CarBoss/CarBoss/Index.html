About The Car Boss

The Car Boss is an a fully functioning WebAPI web application created from scratch. It is composed of a dynamic front-end using AngularJS, HTML5 and Twitter Bootstrap.

It is used to search for cars by using the year, make, model and trim available. The ability to sort and filter results makes finding your car incredibly simple!

This application acceses the NHSA's car recall database using the provided WebAPI to how car recalls

Includes images found using Bing's Search WebAPI. The application dynamically formats itself based on available content / device screen size.
